# Disciplina de Macroevolução

![logo](https://github.com/eknery/disciplina_MFC/blob/main/auxiliar/logo.jpg?raw=true)

## Objetivos 
### Principal: 
Fundamentar a teoria sobre macroevolução e seus principais métodos.

### Específicos:
 - Exemplificar hipóteses macroevolutivas;
 - Explicar o potencial e a limitação de modelos no estudo de macroevolução;
 - Demonstrar a aplicação de métodos para responder hipóteses macroevolutivas;
 - Justificar a escolha de métodos e modelos com base nas hipóteses de interesse;


## Público-alvo: 
Estudantes de pós-graduação em biologia e áreas afins.

## Pré-requisitos: 
A disciplina requer que os estudantes tenham familiaridade com a linguagem de programação R para executar as atividades práticas. Conhecimentos básicos sobre estatística são desejáveis para compreensão do conteúdo teórico.

## Ementa:
Tempo e modo de evolução; modelo de movimento browniano; sinal filogenético; seleção e restrições evolutivas; modelo Ornstein–Uhlenbeck; correlação evolutiva; método dos contrastes filogenéticos; método dos mínimos quadrados generalizados filogenéticos; cadeais de Markov; reconstrução de estados ancestrais;

## Metodologia de aula: 
O curso terá aulas teóricas e práticas. As aulas teóricas serão tanto expositivas quanto dialogadas, envolvendo a leitura prévia de textos e a discussão em sala de aula. As práticas serão exercícios de aplicação de métodos em dados já disponíveis. Todas as aulas práticas necessitarão de computadores com a linguagem de programação R. Conhecimento em estatística é desejável. 

## Método de avaliação: 
Os estudantes serão avaliados com base na sua participação em aula e elaboração do projeto final. O projeto final envolverá a aplicação justificada de métodos para investigar uma hipótese macroevolutiva. O desenvolvimento do projeto dependerá da participação nas aulas práticas.
